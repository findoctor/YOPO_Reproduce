# YOPO_reproduce
Reproduce works of ["You Only Propagate Once: Accelerating Adversarial Training via Maximal Principle" ](https://arxiv.org/abs/1905.00877).

## Prerequisites
- PyTorch 1.0.1
- Python 3
- easydict

## How to run the code
###### PGD training:
> python  MNIST/train_pgd.py

###### YOPO training
> python MNIST/train.py
